# Spring-Security-Demo
Spring Security 在 Spring Boot 中的集成示例



## 说明

* 其中：
  * Simple-Example：简单示例
  * BackDoor：身份验证 - 后门
  * Authentication-DB：身份验证 - 数据库
  * Authorization-DB：访问控制 - 数据库
  * JSON：JSON
  * VerifyCode：验证码
  * JWT：JWT
  * CustomLoginController：自定义登录接口
* 默认登录密码为：pwd
* 个人学习项目
